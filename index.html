📋 ขั้นตอนที่ 2 — เลือกหรือสร้างงาน
หลัง login จะเห็นรายการงาน → เลือกงานที่มีอยู่ หรือกด ✨ สร้างงานใหม่
ตั้งรหัสงาน เช่น dmsc2025
📂 ขั้นตอนที่ 3 — เตรียมและอัปโหลด Excel
ดาวน์โหลดไฟล์ตัวอย่าง กรอกข้อมูลผู้เข้าพัก แล้วกดปุ่ม 📂 โหลด Excel
⚠️ ข้อควรระวังสำคัญ
• ห้ามเปลี่ยนชื่อหัวคอลัมน์ เช่น ห้ามเปลี่ยน ชื่อ - สกุล เป็นอย่างอื่น
• ห้ามลบหรือสลับตำแหน่งคอลัมน์ เด็ดขาด
• ไฟล์ต้องเป็นนามสกุล .xlsx เท่านั้น
• ชื่อไฟล์เปลี่ยนได้ตามต้องการ ไม่มีผล
☁️ ขั้นตอนที่ 4 — บันทึกและแชร์
ตรวจสอบข้อมูลแล้วกด ☁️ บันทึกขึ้น Cloud
จากนั้นกดปุ่ม 📤 แชร์ เพื่อส่งลิงก์ให้ผู้เข้าพัก
💡 เคล็ดลับ
• ถ้ายกเลิกห้อง: ใส่ (ยกเลิกห้อง) ต่อท้ายชื่อในคอลัมน์ ชื่อ-สกุล
• พักคู่: กรอกข้อมูลรูมเมทในคอลัมน์ __1
• อัปเดตข้อมูล: โหลด Excel ใหม่แล้วบันทึกทับได้เลย
🚫 ข้อห้ามที่ต้องปฏิบัติอย่างเคร่งครัด
• ห้ามอัปโหลดหมายเลขบัตรประชาชน (เลข 13 หลัก) โดยเด็ดขาด
• ห้ามอัปโหลดข้อมูลทางการแพทย์ เช่น ผลตรวจ ประวัติโรค ยา
• ห้ามอัปโหลดข้อมูลทางการเงิน เช่น เลขบัญชี บัตรเครดิต
• ห้ามอัปโหลดรหัสผ่านหรือข้อมูลความลับขององค์กร
• ระบบนี้ออกแบบมาสำหรับ ข้อมูลการเข้าพักและรถรับส่งเท่านั้น

🔍 วิธีที่ 1 — รู้รหัสงาน (Event Code)
พิมพ์รหัสงานที่ได้รับจาก Admin เช่น dmsc2025 ในช่องแรก แล้วกด เข้าดู
🔎 วิธีที่ 2 — ค้นหาจากชื่องาน
พิมพ์ชื่องานในช่องที่สอง เช่น ประชุมกรม หรือ ประชุมกอง แล้วเลือกงานจากรายการที่แสดงขึ้นมา
📱 วิธีที่ 3 — สแกน QR Code
ขอ QR Code จาก Admin แล้วสแกนด้วยกล้องมือถือ จะเข้าสู่หน้างานโดยตรงเลย
💡 เมื่อเข้าถึงหน้างานแล้ว สามารถค้นหาชื่อตัวเองได้ที่ช่องค้นหา และกดปุ่มโทรหาผู้ประสานงานได้โดยตรง

📝 วิธีสมัคร Admin

2. เลือกแท็บ สมัครใช้งาน
3. ตั้ง Username และ Password
4. กด สมัครใช้งาน แล้วรอ Super Admin อนุมัติ
✨ วิธีสร้างงาน
1. Login แล้วกด ✨ สร้างงานใหม่
2. ตั้ง รหัสงาน เช่น dmsc2025
3. ใส่ชื่องานและคำบรรยาย → กด สร้างงาน
📂 วิธีอัปโหลดข้อมูล
1. กด 📥 Template เพื่อดาวน์โหลดตัวอย่าง Excel
2. กรอกข้อมูลผู้เข้าพักในไฟล์
3. กด 📂 อัปโหลด Excel แล้วเลือกไฟล์
4. กด ☁️ บันทึกขึ้น Cloud — ผู้เข้าพักจะเห็นทันที

• ห้ามลบหรือสลับคอลัมน์
• ไฟล์ต้องเป็น .xlsx เท่านั้น
